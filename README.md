## General info
This repository contains small tasks resolved in JavaScript code.
They are separated in ten branches (each contains several tasks).
Hope I helped you with some ideas! :)

## Technologies
Project is created with:
* JavaScript ES5

## Setup
To run this project, copy repository url.
Then type next command in GitBash:
```
$ cd <location>
$ git clone <url>
```
For seeing results of tasks, you need to open console in your browser.
